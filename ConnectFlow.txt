7 [Client sends encrypted handshake payload]
       |-- Payload contains:
            - Client version
            - Extension version/hash
            - Mods list/hash
            - Optional password
            - Optional request ID for response callback
       |
       v
8 [Server receives and decrypts handshake data using shared secret]
       |
       v
9 [Server validates:]
       - Client version & extension version against allowed hashes
       - Mods hash & list
       - Password (if required)
       - Reject connection if invalid → send error and disconnect

14 [Client applies initial objects and adjusts local clock]

       |
       v
15 [Server sends incremental object updates/deltas]

       v
16 [Server unlocks edits for client]
